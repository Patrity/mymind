# Screenshots

Drop PNGs here with these exact names (referenced by the root `README.md`). Capture in a clean browser at a sensible width (~1400px), light or dark — just be consistent.

| File | What to capture |
|---|---|
| `documents.png` | **Hero.** Documents page in split or preview mode — tree on the left, a real Markdown doc rendering on the right (bonus: a `::mc-callout`). |
| `command-palette.png` | ⌘K palette open with a query showing grouped results (Documents + Memories). |
| `gallery.png` | Image gallery grid, ideally with the tag filter / a few OCR'd images. |
| `tasks.png` | Kanban board with cards across a couple of columns. |
| `memories.png` | Memories page — a few memory cards with scope/relevance badges (or the review queue). |
| `sessions.png` | A session detail view — header stats (tokens/tools) + the transcript. |
| `clipboard.png` | Clipboard thread with a couple of messages showing machine attribution. |

Tip: trim/scale to a similar aspect ratio so the README grid stays tidy. Keep file sizes reasonable (these get committed).
